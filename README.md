Manga Backend API

Proyecto backend desarrollado con Java 17 y Spring Boot para practicar la construcción de APIs REST siguiendo una arquitectura en capas y buenas prácticas de desarrollo.

Descripción

Esta aplicación consiste en una API REST orientada a la gestión de mangas.
El objetivo del proyecto es reforzar conceptos clave del desarrollo backend como:

Separación de responsabilidades (Controller, Service, Repository)

Persistencia con JPA / Hibernate

Modelado de entidades

Gestión básica de validaciones y control de errores

Organización limpia del código

Actualmente se encuentra implementado el CRUD completo de la entidad principal (Manga), y el proyecto se está ampliando progresivamente con nuevas entidades y relaciones.
